# File Integrity & Digital Signature Tool

## Overview
This project is a Python CLI tool that ensures file integrity and authenticity using SHA-256 hashing and RSA digital signatures.

---

## Features
- Generate SHA-256 hashes for files
- Create a manifest (metadata.json)
- Detect file tampering
- Generate RSA key pairs
- Sign files using private key
- Verify signatures using public key

## Security Concept

- Hashing ensures data integrity
- RSA ensures authenticity and non-repudiation
